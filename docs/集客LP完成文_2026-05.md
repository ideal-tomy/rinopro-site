# 集客LP完成文（AXEON）

## 0. 使い方
- 本文は、現行サイトの掲載内容（`src/lib/content/site-copy.ts`）を土台にしたLP向け完成文です。
- そのまま掲載できるよう、セクションごとに「見出し」「本文」「CTA」をセットで記載しています。
- トーンは「営業圧を抑え、整理と判断を支援する」方針で統一しています。

### 見出し
話すだけで、次が見えてくる。

### 本文

### CTA

### 見出し
こんな状態が続いていませんか？

情報が散らばっていて、確認や引き継ぎに手間がかかる。  
担当者ごとに対応品質がぶれ、判断が遅れる。

いまの課題を完璧に言語化できていなくても問題ありません。  
AXEONは、質問に答える形で、課題の芯を一緒に明確にしていきます。

## 4. AXEONの進め方セクション

### 見出し
小さく検証しながら、実装まで進めます

### 本文（導入）
「作ること」だけを先に決めるのではなく、どこで何を確認し、何を渡すかを明確にしながら進めます。

### ステップ
1. 現状整理  
業務フローと負荷ポイントを可視化し、改善余地を特定します。

2. 要件化  
入力・出力・判定基準を定義し、AIやシステムに任せる範囲を明確化します。

3. 試作・現場検証  
小さく試し、現場のフィードバックを反映しながら実用性を高めます。

4. 本実装  
既存システムとの連携、運用設計、改善ループまで含めて整えます。

### 補助文
各段階で確認ポイントを共有し、仕様の自然増殖を抑えながら進行します。

### CTA
- 開発の流れを見る
- 見積もりで要件を整理する
