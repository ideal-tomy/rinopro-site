## 【10】よくある質問

### セクションタイトル
**09　よくある質問**

# よくある質問

#### Q1. AIに詳しくないのですが、相談できますか？
**A.** もちろんです。むしろ「詳しくない方」のご相談がほとんどです。専門用語は使わず、業務の言葉でお話しします。

#### Q2. 小さな店・会社でも、相談に乗ってもらえますか？
**A.** はい。従業員数や売上規模で断ることはありません。困りごとがあれば、まずはお話だけでも聞かせてください。

#### Q3. どのくらいの費用がかかりますか？
**A.** 内容によって幅があります。すぐに数百万円が必要というケースばかりではありません。月額の顧問契約や、業界別のパッケージから始められる場合もあります。まずは無料相談でお見積もりをお伝えします。

#### Q4. 何から始めればいいか、分かっていないのですが大丈夫ですか？
**A.** 大丈夫です。「何が困っているか」を整理するところから一緒に始めます。具体的な解決方法は、その後で考えれば十分です。

#### Q5. 提案を聞いたあと、断っても大丈夫ですか？
**A.** もちろんです。提案を聞いて「うちには合わない」と思ったら、断って構いません。営業の追いかけ電話やしつこい連絡はしません。

#### Q6. 個人事業主・フリーランスでも相談できますか？
**A.** ご相談自体は可能です。ただし、ご提案できる内容に限りがあるため、まずは状況をお聞かせください。

## 原稿としての補足メモ

### キャッチコピーのバリエーション（A/Bテスト想定）
- A案（採用）：**小さなお店・小さな会社のための、AIのはじめ方。**
- B案：**人が足りない、時間が足りない。AIに、ちょっと手伝ってもらう。**
- C案：**AIは難しい？　でも、いまの仕事は、もっとラクにできます。**

### 中堅・大企業向け版との主な違い
- FVキャッチが「敷居の低さ」全面（規模を強調しない、用語を使わない）
- 課題リストの表現が平易（「人手が足りない」「現場に出っぱなし」など、業務シーンに即した言葉）
- 3つの考え方の見出しが「行動を示す動詞」になっている（「〜しません」が3つ並ぶ）
- 標準体制セクションを「代表が直接お話します」に置換
- ご支援可能領域から価格レンジを削除、対応業種を具体名で列挙
- CTAコピーが「まずは話を聞いてみる（無料）」で統一
- セクション英語サブタイトルを排除し、日本語のみ

### コピーの一貫性チェック
- 専門用語（DX, PoC, AI実装）の不使用 ✓
- 横文字役職を硬く名乗らない ✓
- 「あなた」「気軽に」「大丈夫」など、安心感ある語彙の多用 ✓
- 「断っても大丈夫」「押し売りなし」の明示 ✓
