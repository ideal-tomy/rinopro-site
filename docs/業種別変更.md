#### 実装ヒント

- グリッド: PC `repeat(3, 1fr)` / タブレット `repeat(2, 1fr)` / モバイル横スクロール
- サムネ画像は次世代フォーマット(.webp)推奨
- モバイル横スクロール時、`scroll-snap-type: x mandatory`
- 各カードからのリンクは `target="_blank"` で新タブ(他サイトのデプロイURLなので)
- 社内ナレッジBOTのみ自サイト内リンク
